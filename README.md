# rust_sh
Experimentation with Rust programming language to create a shell for Unix-based systems

## Features:
* exit command
* cd command
* pipes

## Future:
* background process (&)
* signal handling
* redirection

## To Use:
To run the shell(dev):

    cargo run

To build binary:

    cargo build
